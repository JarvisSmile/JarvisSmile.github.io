File_Transfer_between_Local_and_GitHub_based_on_GitBash

Posted on 2018-03-29 | In Notes |
Words count in article: | Reading time ≈

使用GitHub越来越多了，那么，如何传递本地与仓库的数据就成了问题，学习了Git Bash命令行链接，在这里做一个小笔记

本地建仓

  • 创建一个新的文件夹
  • 在文件夹里打开Git Bash 点击右键，选择Git Bash here
  • 输入命令：$git init 这是初始化本地仓库

建立本地仓库与GitHub仓库关联

  • 输入命令：$git remote add origin 仓库链接 比如：git remote add origin https://github.com/JarvisSmile/JarvisSmile.github.io.git

将GitHub仓库下载到本地

  • 输入命令：$git pull origin master

修改内容并提交commit

  • 输入命令：$git status
  • 输入命令：$git add.
  • 输入命令：$git status
  • 输入命令：$git commit -m”这里写你对修改的描述“
  • 输入命令：$git status

将本地修改上传到Github仓库

  • 输入命令：$git push origin master

登陆GitHub仓库，确认已修改
